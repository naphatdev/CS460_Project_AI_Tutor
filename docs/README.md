# docs/

โฟลเดอร์นี้เก็บเอกสารประกอบ project ที่ไม่ใช่โค้ด

## สิ่งที่จะมาเก็บไว้ที่นี่

| ไฟล์ | เนื้อหา |
|---|---|
| `proposal.pdf` | Proposal เสนออาจารย์ตอนต้นเทอม |
| `workflow.png` หรือ `workflow.drawio` | Diagram อธิบาย workflow 6 ขั้น (ดูใน [../README.md](../README.md)) |
| `architecture.png` | Diagram สถาปัตยกรรมระบบ (frontend ↔ backend ↔ OpenRouter / DB) |
| `slides/` | สไลด์ presentation ปลายเทอม |
| `meeting-notes/` | บันทึกประชุมทีม (optional) |

## เอกสารที่ **ไม่ได้** อยู่ในนี้

- โครงสร้าง project — ดู [`../README.md`](../README.md)
- สัญญา API — ดู [`../API_CONTRACT.md`](../API_CONTRACT.md)
- Tech stack + เหตุผลที่เลือก — ดู [`../TECH_STACK.md`](../TECH_STACK.md)

## หมายเหตุ

- ใช้ชื่อไฟล์ภาษาอังกฤษ ไม่มีเว้นวรรค (`workflow.png` ไม่ใช่ `workflow ของระบบ.png`)
- ไฟล์ `.pdf` / `.png` ขนาดใหญ่ ๆ ให้ใช้ Git LFS ถ้าเกิน 50 MB
